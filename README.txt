 2. 各ページの説明
================================================================

【部の方針】
　部の運営方針が表示されます。
　管理者のみテキストの編集・保存が可能です。

【予定表】
　練習やイベントのスケジュールを一覧表示します。
　予定をクリックすると詳細がポップアップ表示されます。
　管理者のみ予定の追加・編集・削除が可能です。

　▼ 予定の色分け
　　赤　：ライブ・イベント当日
　　青　：部会
　　黄　：イベント
　　緑　：提出〆切

【ホール予約】
　ホールの予約状況をカレンダーで確認できます。
　クリックして予約可能な日付を選び、バンド名と入力者名を登録します。
　予約の削除も可能です。

　※ 予約受付は、希望日の「土日祝を除いた4日前の23:59まで」です。
　※ 土日・祝日および受付期限を過ぎた日付はグレー表示され、予約できません。

【部内連絡】
　部内の連絡事項が時系列で表示されます。
　管理者のみ新規投稿・削除が可能です。
　投稿日から6か月が経過した連絡は自動で非表示になります。

【部会出席可否】
　日付と部員名のテーブル形式で出欠を管理します。
　各マスをクリックすると「〇」「×」「空欄」が切り替わります。
　管理者のみ日付の追加・削除が可能です。

　▼ 色のルール
　　〇　… 文字色：黒、背景色：グレー
　　×　… 文字色：赤、背景色：白

【Sajek写真館】
　フォルダ形式で写真を管理できます。
　フォルダ内の写真をクリックすると拡大表示され、ダウンロードボタンから
　画像を保存できます。
　管理者のみフォルダ作成・写真アップロード・削除が可能です。

================================================================
 3. スマートフォン対応
================================================================

　スマートフォンで開くと、画面上部にハンバーガーメニュー（三本線）が
　表示されます。タップするとサイドメニューが開き、ページを切り替えられます。

================================================================
 4. データのリセットについて
================================================================

　データを初期状態に戻したい場合は、ブラウザの設定から
　LocalStorage（ローカルストレージ）を削除してください。

　Chromeの場合: F12 → アプリケーション → ローカルストレージ
　→ 「sajek_data」を右クリック → 削除
